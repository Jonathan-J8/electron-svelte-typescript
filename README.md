# Electron + Svelte + Typescript Template

- electron-builder : https://www.electron.build/
- rollup : https://rollupjs.org/guide/en/

## Script

```
npm i
npm run dev
npm run build
```

## Files Structure

<pre>
.   
├── app                     # Svelte app    
├── build                   # Transpiled files (```npm run dev```)   
├── dist                    # Compiled files (```npm run build```)      
├── proxy                   # Electron app   
├── package.json                      
├── README.md                      
└── rollup.config.js   
</pre>
